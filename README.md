# Aura Agent
 Have you ever wonder what does an ai agent in your terminal or your browser have aura? well thanks to the adk i used its here!
## How does it work
It uses an adk and uses gemini 3.5 flash model
### Install the agent
in your terminal (with that folder)
run:
````powershell
pip install google-adk

````
then run:

````powershell

adk run my_agent

````

or
use the web browser with the tutorial to run the agent in the adk website

that is how u use the agent
